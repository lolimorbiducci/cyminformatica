Software para Resto - Bar

Permite organizar y controlar la atención de los mozos en las mesas, el control de consumo de las mismas, la atención de la barra de tragos, la impresión de comandas en la cocina y el manejo del delivery. Posee además, como modulo optativo, la posibilidad de que cada mozo pueda realizar el pedido mediante una TABLET conectada inalámbricamente al sistema central. El sistema realiza todas las actividades administrativos-contables necesarias para cumplimentar los circuitos administrativos y comerciales de todo el comercio. Posee una interface amigable e intuitiva al operador, niveles de seguridad jerárquicos y un módulo de consultas en bloque o 3D que el propio usuario puede configurar. El sistema tiene una gran adaptabilidad a todas las empresas dedicadas a la actividad gastronómica.

Software para supermercados

El software para Supermercados posee dos módulos operativos: Modulo punto de venta: Es un software desarrollado para línea de cajas. Ágil y seguro, con logueo de cajero. Manejo de cuentas corrientes, tarjetas de crédito y otros medios de pagos. Modulo administrador: Realiza todas las actividades administrativo-contables necesarias para cumplimentar los circuitos administrativos y comerciales de toda la empresa. Posee una interface amigable e intuitiva al operador, niveles de seguridad jerárquicos y un módulo de consultas en bloque o 3D que el propio usuario puede configurar. El Modulo Administrador posee conexión con todos los dispositivos periféricos y con la línea de Cajas.

Software para hospitales

Este software es una herramienta única que permite un registro y seguimiento del plan de vacunación de cada ciudadano a lo largo de su vida. Las fichas individuales de cada persona poseen la información necesaria para que el propio sistema pueda emitir automáticamente correos electrónicos advirtiendo la falta de una o varias dosis de vacunas. Posee informes para la provisión de dosis de vacunas a aplicar, informes de vacunas aplicadas, y toda aquella información adicional que es requerida por la región sanitaria. Permite emitir el certificado oficial de vacunación de cada persona, hacer campañas de vacunación en los colegios y conocer los datos demográficos de la población en la región de influencia del hospital. El Sistema P.N.V. posee como principal objetivo, no solo el control cuantitativo del área de vacunación, sino el control cualitativo de cada persona registrada, pudiéndose individualizar y contactar con cada una de ellas.
